# Documentación Funcional y Contexto

Esta carpeta es el repositorio central de Conocimiento Funcional del negocio. 

Por favor, deposita aquí:
* Especificaciones Funcionales o de Producto (PRD).
* Historias de Usuario y Criterios de Aceptación.
* Diagrámas de flujo o reglas de negocio críticas.

**Flujo Sugerido:** 
1. Arrojas un documento aquí (Ej: `login-especificaciones.md` o `.txt`).
2. Luego, solicitas la generación de *Casos de Prueba* detallados basados estrictamente en esas reglas, los cuales se volcarán en la carpeta vecina `/plan-de-pruebas`.
